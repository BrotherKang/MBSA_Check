# MBSA_Filter

此工具是為了方便將單位內進行檢測的電腦作統計用
因為微軟工具檢測產出後，有太多資訊包在裡面且一台電腦一個檔案
要一一點開檔案太麻煩了
就寫個小工具撈出需要的部分查看
